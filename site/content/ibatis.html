iBatis

The iBATIS Data Mapper framework makes it easier to use a database with Java applications. iBATIS couples objects with stored procedures or SQL statements using a XML descriptor. Simplicity is the biggest advantage of the iBATIS Data Mapper over object relational mapping tools.

The iBatis Gps Device can index the data mapped by select statements (in the XML configuration files). It uses OSEM to map from the result objects to the search engine.
